# Context

This is a webpage for a mechanical workshop (auto repair shop) that provides information about the services offered, the team of mechanics, and contact details. The webpage is designed to be user-friendly and informative for potential customers looking for automotive repair services. The company is called "Monkey King Garage" and is located in Puerto Montt, Chile.

## Technology stack

The webpage is built using modern web technologies to ensure a responsive and engaging user experience. The technology stack includes:

- Vue3 With Vite.
- Vue-router
- Pinia
- Vuetify
- PNPM (Do not use npm or yarn)
- typescript
- Git and Github

## Important considerations

- Make the code modular, reusable, simple and readable. Remember that the code should be easy to maintain and extend in the future, so the simplest solution is often the best one.
- If we add or edit something new that is important for the whole context of the project, feel free to add it /eddit it on this `AGENTS.md` file, so we can keep track of the important information about the project.
- Prefer Vuetify components over custom components when possible, as they are well-tested and provide a consistent look and feel.
